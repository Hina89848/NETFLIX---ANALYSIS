Netflix Data Analysis Using Python
Tools Used:

Python, Pandas, NumPy, Matplotlib

Overview:

This project analyzes the Netflix dataset to explore the distribution of Movies and TV Shows, top directors, popular genres, and trends over the years.

Objectives:

Clean and preprocess the dataset

Check null values

Show records and structure

Visualize data using bar charts

Find top 10 directors with most titles

Explore TV Shows vs Movies count

Key Insights:

Netflix has more Movies than TV Shows

Some directors created the highest number of titles

Popular genres and release year patterns explored
